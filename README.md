# forge-reality-centos-deployment
Website Deployment on CentOS Using Apache

Overview

This project demonstrates the deployment of a static HTML website on a CentOS Stream 9 virtual machine using Apache HTTP Server.

Technologies Used

- CentOS Stream 9
- Apache HTTP Server (httpd)
- VirtualBox
- Vagrant
- Git Bash
- HTML
- CSS
- JavaScript

Project Highlights

- Configured a CentOS virtual machine.
- Installed and configured Apache HTTP Server.
- Deployed a static HTML website.
- Verified website accessibility through the VM network.
- Gained hands-on experience with Linux server administration and web hosting.
